## Getting Started

### Prerequisites

- Node.js and npm installed on your machine.

### Installation

1. Clone this repository:

```bash
git clone https://github.com/yourusername/your-nextjs-app.git
cd your-nextjs-app


### Installing packages
npm i
# or
npm install


### Installation

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

##The app will be available at http://localhost:3000.
